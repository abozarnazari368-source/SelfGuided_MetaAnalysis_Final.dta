# Effectiveness of Self-Guided Digital Interventions for Suicide Prevention:
## A Systematic Review and Meta-Analysis of Randomized Controlled Trials

This repository contains the Stata code and dataset used to perform the meta-analysis.

### Software
Stata version 17 (StataCorp, College Station, TX, USA)

User-written packages:
metan  
metareg  
metabias  

### Folder structure
data → dataset used in the meta-analysis  
code → Stata do-file for all analyses  
outputs → generated figures  

### Reproducibility
To reproduce all analyses and figures, run:

do code/meta_analysis.do
